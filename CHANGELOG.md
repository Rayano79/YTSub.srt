# Changelog

All notable changes to YTSub.srt will be documented in this file.

## [1.0.1] - 2025-11-11

### Added
- Smart video ID detection from filename, URL, or active tab
- Subtitle library with video titles and upload dates
- Enable/Disable toggle in extension header
- Font size slider (10-40px range)
- Text shadow with adjustable opacity
- Background with adjustable opacity
- Intelligent subtitle positioning for all viewing modes
- RTL language auto-detection
- Responsive subtitle positioning with multiple observers
- Clean, compact UI (320x480px)

### Fixed
- Subtitle positioning in normal mode (centered correctly on initial load)
- Subtitle alignment issues when switching between fullscreen and normal mode
- Extra black space in subtitle background
- Status message area showing empty white space

### Changed
- Rebranded from RaySubYT to YTSub.srt
- Improved UI design with better spacing
- Optimized popup layout to prevent scrolling
- Updated all CSS and JS IDs to match new branding

## [1.0.0] - Initial Release

### Features
- Basic SRT subtitle upload
- YouTube video integration
- Simple subtitle display
